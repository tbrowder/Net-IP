# Net::IP

This module is a beginning port of the Perl module `Net::IP` to the
`Raku` programming language.

## Synopsis:


```
#!/usr/bin/env perl6

use Net::IP;

# manipulate IP addresses...
```

See the internal documentation in the terminal window by entering:

```
$ p6doc Net::IP
```
